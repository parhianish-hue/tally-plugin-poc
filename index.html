Import Amazon Data into TallyPrime

This plug-in allows you to import Amazon data (Excel) into TallyPrime

* Compatible from TallyPrime 7.1 onwards
The plugin is now enabled -> Follow instructions provided on the right hand side to import Amazon data.

1) Enter serial number

2) Click Get Plugin

3) Enter OTP

4) Plugin is enabled successfully

5) Activate plugin in TallyPrime by restarting or by updating license

Instructions to import Amazon data
1. Pre-Process the Amazon Excel File
Use Amazon data processing tool from Amazon folder in Tally installation path to pre-process the data
Optional: Import Pre-requisite ledgers (One time)
Import required ledgers if not available
2. Import Transactions in TallyPrime
Import transactions with required mapping template
