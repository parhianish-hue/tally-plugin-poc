Import Amazon Data into TallyPrime

This plug-in allows you to import Amazon data (Excel) into TallyPrime

* Compatible from TallyPrime 7.1 onwards
The plugin is now enabled for the serial number provided → Follow the instructions on the right

1) Enter serial number

2) Click Get Plugin

3) Enter OTP received on your tally registered email-id

4) Plugin activates automatically

Instructions

Enable Plugin in TallyPrime
Restart TallyPrime or update license to use the plugin
Pre-Process the Amazon Excel File
Use .xlsm file inside Amazon folder to preprocess data
Import Ledgers
Import ledgers and map SKU as alias(One-time)
Import Transactions in TallyPrime
Use Excel import with correct mapping template
Details given in Read Me file in Amazon Folder
